# Draggable Ball

A simple HTML page with a draggable ball that has gravity.

## Local Development

Open `index.html` in your browser.

## Deployment

This project is set up for deployment on Render as a static site.

1. Create a new repository on GitHub.
2. Push this code to your GitHub repository.
3. Go to [Render](https://render.com) and create a new account if you don't have one.
4. Create a new Static Site.
5. Connect your GitHub repository.
6. Set the build settings:
   - Build Command: (leave empty)
   - Publish Directory: .
7. Deploy!

The site will be live at the URL provided by Render.
